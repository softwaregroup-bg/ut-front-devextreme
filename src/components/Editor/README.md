# Editor

## How to use

```jsx
import Editor from 'ut-front-devextreme/core/Editor';

<Editor className='name' />;
```

## Props

- **className** - class applied to root `Editor` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
